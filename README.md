# fantasy-analytics

Idea:
Using ScraperFC, we scrape FBRef data. Using this data, we can analyze which players perform best regarding fantasy league points.

There are a few interesting types of data:
- Players
- Teams
- Competitions
- Seasons
- Positions
- Matches
